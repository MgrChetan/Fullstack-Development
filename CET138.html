What is Full Stack Development?

Full stack development refers to the development of both front-end (client-side) and back-end (server-side) portions of a web application.

Full Stack Development 3 Main Components:

a. Frontend Development

Frontend development is the practice of producing HTML, CSS and JavaScript for a website or web application so that a user can see and interact with them directly.

Examples: ReactJS, VueJS, NextJS

b. Backend Development

Backend development refers to the server-side of an application and everything that communicates between the database and the browser.

Examples: PHP, NodeJS, Laravel, NestJS

c. Database Development

Database development is the process of designing, implementing and maintaining a database system to meet user requirements.

Examples: MongoDB, MySQL, Prisma, DuckDB, Redis

Full Stack Development Frameworks

Full stack development relies on multiple frameworks for the best efficiency of the web application. These frameworks provide a structured way to build applications with pre-built components and patterns.

Some of the popular full stack development frameworks are:

  • MERN Stack
  • MEAN Stack
  • LAMP Stack
  • JAM Stack
  • Python + Django + SQLite
MERN STACK

MERN Stack is a JavaScript stack that makes deployment easier. It's a collection of technologies that enables faster application development.

MongoDB

A document-oriented NoSQL database used for high volume data storage.

Express JS

A web application framework for Node.js designed for building web applications and APIs.

React JS

A JavaScript library for building user interfaces, particularly single-page applications.

Node JS

A JavaScript runtime built on Chrome's V8 JavaScript engine for building server-side applications.

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications.

Sections Used in HTML:

a. Header

The header represents a container for introductory content or a set of navigational links.

b. Body

The body contains the content of the document, such as text, images, links, etc.

c. Footer

The footer defines a footer for a document or section, typically containing authorship, copyright, etc.

Common HTML Tags:

Tag Description
<a> Defines a hyperlink
<hr> Defines a thematic break or horizontal rule
<li> Defines a list item
<ul> Defines an unordered list
<ol> Defines an ordered list
<table> Defines a table

Implementing CSS in HTML

CSS can be added to HTML in three ways: inline, internal, and external.

/* External CSS */
<link rel="stylesheet" href="styles.css">

/* Internal CSS */
<style>
  body { font-family: Arial; }
</style>

/* Inline CSS */
<p style="color: blue;">This is a blue paragraph.</p>

Implementing JavaScript in HTML

JavaScript can be added to HTML in similar ways: inline, internal, and external.

/* External JS */
<script src="script.js"></script>

/* Internal JS */
<script>
  function myFunction() {
    alert('Hello World!');
  }
</script>

/* Inline JS */
<button onclick="alert('Clicked!')">Click me</button>

Implementing Images and Iframes

/* Image */
<img src="image.jpg" alt="Description" width="500" height="600">

/* Iframe */
<iframe src="https://www.example.com" title="Example" width="400" height="300"></iframe>

Implementing CDN Links

CDN (Content Delivery Network) links allow you to include external libraries without hosting them yourself.

/* Bootstrap CDN */
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

/* Font Awesome CDN */
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages — for example, to alter the font, color, size, and spacing of your content.

Different Types of CSS

Inline CSS

Applied directly to HTML elements using the style attribute.

<p style="color: blue;">
  This is blue text.
</p>
Internal CSS

Defined in the <head> section of an HTML page.

<style>
  p { color: blue; }
</style>
External CSS

Defined in an external .css file and linked to the HTML.

<link rel="stylesheet" href="styles.css">

Selectors, Classes, and IDs

/* Element selector */
p { color: blue; }

/* Class selector */
.my-class { font-weight: bold; }

/* ID selector */
#my-id { background-color: yellow; }

Core CSS Features

Colors, Fonts, Margins/Padding, Borders
.example {
  color: #ff0000; /* Red color */
  font-family: Arial, sans-serif;
  margin: 10px; /* Outer spacing */
  padding: 15px; /* Inner spacing */
  border: 1px solid black;
  border-radius: 5px;
}
Layout with Flexbox
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
Layout with Grid
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}
Responsive Design with Media Queries
/* Default styles */
.container {
  width: 100%;
}

/* Styles for screens 768px and wider */
@media (min-width: 768px) {
  .container {
    width: 750px;
  }
}

/* Styles for screens 992px and wider */
@media (min-width: 992px) {
  .container {
    width: 970px;
  }
}

JavaScript

JavaScript is a programming language that enables interactive web pages and is an essential part of web applications.

JavaScript Fundamentals

Variables
// Variable declarations
var oldVariable = "This is the old way";
let changeableVariable = "This can be changed";
const constantVariable = "This cannot be changed";
Functions
// Function declaration
function greet(name) {
  return "Hello, " + name + "!";
}

// Arrow function (ES6)
const greet = (name) => {
  return `Hello, ${name}!`;
};
Conditionals
let temperature = 25;

if (temperature > 30) {
  console.log("It's hot outside!");
} else if (temperature > 20) {
  console.log("It's warm outside!");
} else {
  console.log("It's cool outside!");
}
Loops
// For loop
for (let i = 0; i < 5; i++) {
  console.log("Iteration: " + i);
}

// While loop
let i = 0;
while (i < 5) {
  console.log("Iteration: " + i);
  i++;
}

document.getElementById("myElement").style.color = "blue";
Event Handling
// Inline event handling
<button onclick="myFunction()">Click me</button>

// Using addEventListener
document.getElementById("myButton").addEventListener("click", function() {
  alert("Button clicked!");
});

Practical JavaScript Features

Form Validation
function validateForm() {
  let name = document.getElementById("name").value;
  if (name == "") {
    alert("Name must be filled out");
    return false;
  }
  return true;
}
Simple Animation
function animateElement() {
  let element = document.getElementById("animatedElement");
  let pos = 0;
  let id = setInterval(frame, 10);
  function frame() {
    if (pos == 350) {
      clearInterval(id);
    } else {
      pos++;
      element.style.top = pos + 'px';
      element.style.left = pos + 'px';
    }
  }
}
